OOPS stand for object oriented Programming. It is a concept to understand real
life in the computer world and solve the real life problems using or by creating the objects. If any language support oops concept that programming language is 
known as Object Oriented Programming Language. OOPS has following paradigms.
1. Class: Class gives the theoratical description. It does not exist in the world. It is just a blue print or structure or imagination.
2. Object: Anything which exist in the world and can be describe is known as 
object. It may be a person , a place or a thing.
3. Relationship between a class and object : Class gives the theoratical description 
about an object it tells what type of an object is and object is real life 
representative of the class.   
4. Encapsulation : It is a process to wrap the information about an object 
    which you want to create into a single unit. After encapsulation the encapsulated unit is known as class. During encpsulation we wrap the 
    4.1 Properties: Decide how an object look like
    4.2 Functions: Decide how an object work
    4.3 Events : What type of action an object can take
    4.4 Event Handler: Associate a function with an event then associated function known as event handler. 
    During Encapsulation we also decide which one member of the class should be
    private or public so to make a member private is known data data hiding.
